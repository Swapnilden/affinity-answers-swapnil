# affinity-answers-swapnil

This repository contains my solutions for the recruitment assessment.

## Structure
- `q1_product_scraper.py`: Python script to scrape product info from MDComputers.
- `q2_database_queries.sql`: SQL queries for the Rfam taxonomy questions.
- `q3_company_sorter.sh`: Shell script to parse and sort S&P 500 data.

## How to Run

### Question 1: Python Scraper
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the script:
   ```bash
   python q1_product_scraper.py
   ```

### Question 2: SQL
The queries in q2_database_queries.sql are written for the Rfam public database.

### Question 3: Shell Script
1. Make the script executable:
   ```bash
   chmod +x q3_company_sorter.sh
   ```
2. Run the script:
   ```bash
   ./q3_company_sorter.sh
   ```
